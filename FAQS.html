Frequently Asked Questions

What is Coding?

Coding is what makes it possible for us to create computer software, apps and websites. Your browser, your OS, the apps on your phone, Facebook, and this website – they’re all made with code.

Is coding hard?

No coding is not hard at all. You really do not need any previous experience or knowledge of programming before learning to code. The best way people learn is to do it themselves and today there are many free resources online available to help you get started.

What language should I start with?

Html and css are not programming languages but mark-up languages.

Why does coding look difficult?

Because it's different from what you are used to. All the numbers, codes, signs might look scray at first, but no need to worry. Once you start learning about coding it will get easier to understand all those data. You will never know if you like something until you try it.
